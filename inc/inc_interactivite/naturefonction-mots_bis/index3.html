Connais-tu la nature et la fonction des mots dans la phrase ?

  • La nature d’un mot est son identité. Elle ne change jamais.
  • La fonction d’un mot est son rôle dans une phrase. Elle peut changer selon le contexte.

→ Mon est un déterminant possessif.
Il introduit le nom « chat » et précise (détermine) de quel chat il s’agit.

→ chat est un nom commun. Il est le cœur du groupe nominal « Mon chat ».
Mon chat est le sujet du verbe « dort » → Qui dort ? C’est mon chat.

→ dort est un verbe conjugué.
Le verbe est le noyau de la phrase. Il définit l'action ou l'état du sujet « Mon chat ».
Pour reconnaitre un verbe, on change le temps afin de modifier la conjugaison.
→ Mon chat dort. → Mon chat dormait.

→ sur est une préposition. Elle introduit une précision.
   → Où est le chat ? Sur le canapé.

→ le est un déterminant défini.
Il introduit et détermine le nom « canapé ».

→ canapé est un nom commun. Il est le cœur du groupe nominal « le canapé ».
→ sur le canapé vert est un complément circonstanciel de lieu : il indique où se trouve le chat.
Le rôle du complément circonstanciel est de décrire les circonstances, c'est-à-dire le contexte, dans lequel se déroule l'action du verbe.

→ vert est un adjectif qualificatif : il précise (qualifie) de quelle couleur est le canapé.

→ il est un pronom personnel. Il remplace le groupe nominal « Mon chat ».
il est le sujet du verbe « ronronne ».

→ ronronne est un verbe. Il exprime l'action de « il ».

→ de est une préposition qui relie et accroche le verbe « ronronne » au nom « plaisir ».

→ plaisir est un nom commun. On peut aussi le faire précéder d'un déterminant. Ici, il complète l'action du verbe « ronronne ».
